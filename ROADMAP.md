# CollabAI 开发路线图

## v1.1 ✅ 已发布

树形组织拓扑 + 子组协调 + 结构化任务消息 + 35 项测试全部通过。

## v1.2 计划中

### 优先级 1：AI 深度集成

| 功能 | 说明 | 状态 |
|------|------|------|
| Gateway AI 处理 task | Gateway 收到 task 消息时调用 AI 分析并生成回复 | 🔲 |
| 自动变更检测 | AI 检测到接口变化 → 自动查找受影响用户 → 发送通知 | 🔲 |
| 组内 AI 聚合 | 子组 AI 定期汇总成员活动向上汇报 | 🔲 |

### 优先级 2：权限与安全

| 功能 | 说明 | 状态 |
|------|------|------|
| Room 角色执行 | admin/developer/viewer 权限在 API 层生效 | 🔲 |
| 记忆访问控制 | 敏感记忆仅特定角色可读 | 🔲 |
| Gateway 速率限制 | 防止 API 滥用 | 🔲 |

### 优先级 3：开发者体验

| 功能 | 说明 | 状态 |
|------|------|------|
| 配置文件优化 | collab-ai.json 支持更多自定义选项 | 🔲 |
| 会话管理增强 | 批量导出、合并会话、标签分类 | 🔲 |
| 错误诊断 | LLM 错误自动分析 + 建议修复 | 🔲 |

## v1.3 计划中

### 插件系统

| 功能 | 说明 | 状态 |
|------|------|------|
| 插件自动加载 | 扫描目录自动 registerTool | 🔲 |
| 插件市场 | 社区共享工具包 | 🔲 |
| 自定义 Provider | 第三方模型接入接口 | 🔲 |

### 智能检索

| 功能 | 说明 | 状态 |
|------|------|------|
| 向量语义搜索 | 记忆 >1000 条后替代 SQLite LIKE | 🔲 |
| 代码库索引 | 项目代码自动索引供 AI 检索 | 🔲 |
| 相似记忆推荐 | 记录记忆时自动推荐相关条目 | 🔲 |
